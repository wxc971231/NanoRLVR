# NanoRLVR

Minimal RLVR (Reinforcement Learning with Verification Rewards) implementation.

## Structure

- `train_rlvr.py`: Main entry point for training.
- `rl/`: Contains RL algorithms, rollout logic, and verifiers.
- `scripts/`: Helper scripts for data and testing.
- `data/`: Dataset storage.
- `checkpoints/`: Model checkpoints.
- `runs/`: Logs.
